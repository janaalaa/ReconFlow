ReconFlow Documentation 📖

Official setup and usage guide for the ReconFlow automation framework.

1 Installation

Clone the repository from GitHub to your local machine:

git clone https://github.com/janaalaa/ReconFlow.git
cd ReconFlow

2 Usage

To start a new reconnaissance mission, use the run action with the domain flag:

# Execute the CLI tool

python3 cli/hunt.py run -d target.com
Action: run

Starts the automated scanning workflow.

Flag: -d

Specifies the target domain for recon.

3 Requirements

Ensure you have the following tools installed and in your system PATH:

  • Subfinder
  • Assetfinder
  • HTTPX
  • Nuclei
